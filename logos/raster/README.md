# 📸 Raster Logos Directory (`/logos/raster/`)

This directory contains pixel-based, high-resolution static logos. These files are optimized for print materials, slide presentations, community documents, and general social media use where vector rendering is unsupported[cite: 16, 17].

## 📁 Allowed File Formats
* **`.png` (Portable Network Graphics):** Must be exported with an alpha-channel **transparent background**. No solid white or black background boxes are permitted unless explicitly part of a stylized badge.

## 📐 Resolution Standards
To prevent blurry or pixelated branding, export at these minimum dimensions from Figma[cite: 13, 19]:
* **Standard Square Icon:** `512 x 512 pixels`
* **High-Res Display Asset:** `1024 x 1024 pixels`

## 🏷️ Asset Naming Standard
* `nexuscore-logo-raster-dark.png` [cite: 18]
* `nexuscore-logo-raster-light.png`
