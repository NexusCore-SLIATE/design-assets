# 📸 Raster Logos Directory (`/logos/raster/`)

Place all pixel-based, high-resolution static assets here.

### 📁 Expected File Formats
* `.png` (Portable Network Graphics) - Must have an alpha-channel **transparent background**. No solid white or black background boxes allowed.

### 🏷️ Naming Standard
`nexuscore-logo-[variant]-[resolution].png` (e.g., `nexuscore-logo-light-512x512.png`)
